# 彩色路径（colorful_paths）

## 题目描述

给定一个有向无环图，包含 $N$ 个节点（编号 $1 \sim N$）和 $M$ 条有向边。每个节点 $i$ 有一个颜色 $c_i$，其中 $c_i = 0$ 表示蓝色，$c_i = 1$ 表示红色。

一条路径定义为节点序列 $v_0, v_1, \dots, v_k$，满足 $v_0 = 1$，$v_k = N$，且对于 $0 \le i < k$，存在有向边 $(v_i, v_{i+1})$。路径是**合法的**当且仅当路径中任意相邻两个节点，若前一个为红色（$c=1$），则后一个必须为蓝色（$c=0$）。特别地，如果路径仅包含一个节点（即 $1 = N$），则认为该路径合法。

你需要计算从节点 $1$ 出发到达节点 $N$ 的合法路径总数。因为答案可能很大，请输出答案对 $10^9+7$ 取模的结果。

## 输入格式

从标准输入读入。

第一行包含两个整数 $N, M$。

第二行包含 $N$ 个整数 $c_1, c_2, \dots, c_N$，表示每个节点的颜色。

接下来 $M$ 行，每行包含两个整数 $u, v$，表示一条从 $u$ 到 $v$ 的有向边。保证图中没有重边，且不存在自环。

## 输出格式

输出一个整数，表示合法路径的数量对 $10^9+7$ 取模的结果。

## 样例

### 样例 1


5 5
1 0 1 0 0
1 2
2 3
2 4
3 5
4 5


对应输出应为 `3`。

解释：合法路径为 1→2→3→5、1→2→4→5、1→2→3→5？实际上还有 1→2→4→5 等，共三条。

## 数据范围与提示

- $1 \le N \le 10^5$
- $0 \le M \le 2\times 10^5$
- $1 \le u, v \le N$，$u \ne v$
- 保证输入的图是有向无环图（DAG），且至少存在一条从 $1$ 到 $N$ 的路径。
- 子任务见后。

子任务划分：
1. （30 分）$N \le 10$，$M \le 45$。
2. （50 分）$N \le 1000$，$M \le 5000$，且对于所有边 $(u, v)$ 均满足 $u < v$（即节点编号已是拓扑序）。
3. （100 分）无额外限制。

提示：注意状态设计和取模，以及如何高效处理拓扑排序。
